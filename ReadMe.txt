Game of Life - Dirk Meyer

Installation:

Da es sich bei dieser Abgabe um eine WebApp handelt die ausschließlich in HTML, CSS und JavaScript geschrieben ist, ist keine Installation notwendig. Sie benötigen zum ausführen eine Internetverbindung (da Skripte und Stylesheets aus dem Internet eingebunden werden müssen.)


Ausführen:

Führe Sie die im Hauptordner enthaltene GameOfLife.html-Datei mit einem Doppelklick in einem Browser Ihrer Wahl aus. Folgen Sie anschließend den Instruktionen und Hinweisen auf der Webseite.


Sprachen:
Die WebApp ist in deutscher Sprache verfasst. Andere Sprachen stehen nicht zur Verfügung.
Der Code ist jeweils in englischer Sprache kommentiert.


Genutzte Programmmiersprachen:
HTML, JavaScript, CSS
